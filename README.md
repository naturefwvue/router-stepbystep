 # 项目名称
记录学习路由的脚本  

# 功能
属性Vue的路由，实现以下几个功能：  
1、导航路由，单页的。  
2、左侧菜单路由，子路由。  
3、tab标签的路由，名称视图。  

# 内容
用博客来学习路由。  
# 文件说明
 index.html 博客页面，演示路由。  
 router1.html 路由的demo，根据官网整理。  
 https://router.vuejs.org/zh/  

# 安装要求
需要有vue.js、vue-router.js文件，这里有提供。  
需要谷歌内核浏览器，IE还是算了，不支持。  
# 使用方式
直接运行即可，也可以建立站点，  
不用各种配置，简单方便。
